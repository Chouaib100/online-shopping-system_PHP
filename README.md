# Khan Store - Boutique E-commerce en PHP/MySQL

Une plateforme e-commerce complète développée en **PHP natif**, **MySQL** et **Bootstrap**, avec gestion du panier, authentification utilisateurs, système de paiement PayPal, administration basique et design responsive.


## Fonctionnalités principales

- Page d'accueil dynamique avec carrousel, produits mis en avant et catégories
- Catalogue produits avec filtres (catégories, marques, recherche, prix)
- Page détail produit avec galerie photos, avis et produits similaires
- Système de panier persistant (connecté ou invité via IP)
- Inscription / Connexion utilisateurs
- Paiement via **PayPal** (redirection + confirmation)
- Gestion basique des commandes (enregistrement après paiement)
- Interface responsive (mobile-first avec Bootstrap)
- Barre de navigation avec dropdown panier et compte utilisateur

## Technologies utilisées

- **Backend** : PHP 7/8, MySQLi
- **Frontend** : HTML5, CSS3, Bootstrap 3/4, jQuery
- **Bibliothèques JS** : Slick Slider, noUiSlider (filtre prix)
- **Paiement** : Intégration PayPal Standard (IPN simulé)
- **Base de données** : MySQL (fichier .sql fourni)
- **Autres** : Font Awesome, AJAX pour chargement dynamique

## Prérequis pour l'installation

- Serveur web : Apache / Nginx
- PHP ≥ 7.4
- MySQL / MariaDB ≥ 5.7
- Navigateur moderne (Chrome, Firefox, Edge...)
